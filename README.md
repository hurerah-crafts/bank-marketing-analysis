# Bank Marketing Campaign Analysis

An end-to-end data analytics project on a Portuguese bank's marketing campaign data (11,162 customers). The project covers data exploration, visualization, SQL-based querying, and dashboarding to understand customer behavior and campaign performance, using **Python**, **SQL**, and **Power BI**.

## Structure
```
python/analysis.ipynb    -> data exploration + charts
sql/analysis_queries.sql -> SQL queries
sql/sql_results.md       -> query results
data/bank.csv            -> dataset
```

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- SQL Server (T-SQL)
- Power BI
